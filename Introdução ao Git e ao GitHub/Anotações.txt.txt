O Git Bash é um terminal extendido para otimizar o uso do Git.

Criação e atualização de  um repositório no Git/GitHub, onde utilizamos o Git Bash, onde foi revisado os principais comandos do Git:

git clone: clona um repositório que foi criado no GitHub e trazemos para o repositório local

git add . :prepara os arquivos que foram modificados no repositório local para ser enviado para o repositório Remoto.

git status : verifica se houve modificações no repositório

git commit: criar um comentário no arquivo modificado que será útil para entender o que foi alterado    

git push origin main; envia as alterações do repositório local para o repositório remoto(GitHub). 

git pull : é usado para buscar e baixar conteúdo de repositórios remotos e fazer a atualização imediata ao repositório local para que os conteúdos sejam iguais
